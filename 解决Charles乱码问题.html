解决Charles乱码问题

发表于 2017-05-22   |   分类于 技巧   |   热度 ℃
| 字数统计 433 字 | 阅读时长 2 分钟

在网上查了半天说在Info.plist文件加字符串就好了，其实没有用，下面有一个方法亲试可以解决乱码问题

安装ssl证书

3.10版本之前的需要去http://www.charlesproxy.com/ssl.zip 下载 CA 证书文件，然后双击 .crt 文件，选择‘总是信任’按钮，在钥匙串访问中即可看到添加成功的证书。

我是4.02版本的，在3.10版本之后的，操作如下：

  • 先点proxy中的macOS Proxy ，如果点击网页，Charles没有出现东西的话，把翻墙软件关了

  • 然后点击help中的SSL Proxying如下图：

  • 点击之后在搜索框中输入Charles，出现（我是已经改好了，原来的话是红色的）：

  • 双击那个证书，改成如下图所示，然后保存：

  • 接下来点击proxy中的SSL Proxying Settings，出现如下图所示，然后点add 在Host里填 * 号 ，在Port里填443，然后点ok：

  • 接下来，你会惊奇的发现，乱码没有了\（￣︶￣）/

注意！！

如果你开了 vpn，先要把 vpn 关掉，还有可能手机调试的时候Charles的左边会出现unknown，这个时候你需要在手机安装证书，步骤如下：

  1. 点击 help -> SSL Proxying -> install Charles Root Certificate on a Mobile…
  2. 点击完会出现
  3. 手机和电脑处于同一个局域网，查看电脑的 ip 地址和查看 Charles 的端口

  4. 在手机 wifi 上填写 http 代理

  5. 在ios手机上的 safari 浏览器上输入 chls.pro/ssl ,然后跟着安装证书
  6. 安装证书之后，在手机上的设置-> 通用-> 关于本机 -> 证书信任设置-> 点击信任你刚才安装的证书
  7. 重启 Charles，完😀
